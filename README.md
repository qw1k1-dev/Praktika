Краткая последовательность:
Скачать ISO OpenMediaVault с официального сайта.
Создать виртуальную машину в VirtualBox (Linux/Debian 64-bit, 2 ГБ RAM, 20 ГБ диск).
Настроить сетевой адаптер в режиме «Сетевой мост».
Загрузиться с ISO, выполнить установку.
После перезагрузки получить IP-адрес и войти в веб-интерфейс (admin / openmediavault).
Добавить и инициализировать дополнительный диск для данных.
